## What is this?

**CoffeePotSocial** is an open source, Twitter-like social network built live, in public, with AI assistance, just because.

---

## The Origin Story

Someone on the internet suggested you couldn't build a real social network using AI.

This repository is the response to that comment.

The entire build is being documented publicly through video. If you'd like to watch the process, follow along, or just enjoy the schadenfreude — the videos are the place to start.

📺 **[Watch the build series →](https://www.youtube.com/@hyper_text_coffee_pot)** *(start here before diving into the code)*

## Contributing

This project is open source and contributions are genuinely welcome — from seasoned engineers to developers just getting started.

**Before you open a PR:**

1. 📺 Watch at least one video from the build series so you understand where we're headed
2. 📖 Read [CONTRIBUTING.md](CONTRIBUTING.md)
3. 🔍 Check open issues for something to work on
4. ☕ Make a coffee

We use a PR-based workflow. Nothing goes directly to `main`. All contributions go through review — not because we don't trust you, but because that's just good engineering.

Yes, this project uses AI assistance in its development. That's kind of the whole point.

AI is a tool. Like a good coffee maker — it doesn't write the code for you any more than an espresso machine makes you a barista. But it does make the whole process considerably faster and more caffeinated.
